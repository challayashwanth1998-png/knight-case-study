Design Decisions

Key choices and rationale

Content-Based Classification

Never uses filenames — only content. Handles "scan001.pdf" correctly every time.

Hybrid Extraction

Excel/CSV use Python regex at $0. Only CDL images use Gemini Vision. Saves ~70% on API costs.

Parallel Processing

ThreadPoolExecutor · 4 workers · ~45s vs ~3min sequential.

Configurable Rules

27 rules in DB with severity + enable/disable. No code changes to tune.

Cross-Document Conflicts

Validates counts, names, FEIN/DOT, duplicate CDLs and VINs across documents.

Team Routing

Auto-routes to Standard, Specialty, Driver, Ops, or Senior UW based on rules.

Dual Intake

Web upload + IMAP email. Agents don't need a portal login.

Database Portability

SQLAlchemy ORM. SQLite → PostgreSQL with one connection string.

Security & Governance

Audit trail · Encryption · Human-in-the-loop

Complete Audit Trail

Every action logged: uploads, classification, extraction, AI, rules, decisions, reviews.

Data Encryption

  • At rest: S3 AES-256
  • In transit: IMAP TLS, Gemini HTTPS
  • Creds: Env vars, never in git

Human-in-the-Loop

No auto-approval. Must Approve, Reject, or Override with notes.

AI Transparency

Tracks input/output tokens, cost, call count per submission.

Scalability Path

  • DB: SQLite → PostgreSQL
  • Compute: EC2 → ECS/Fargate
  • AI: Gemini auto-scales
  • Queue: SQS/Redis for async
