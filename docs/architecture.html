📋 Document Types Processed

📝 Insurance Application

Company info, FEIN, DOT#, MC#, operations, commodities, coverage requested, prior insurance, loss history

🚛 Driver Roster

Driver names, DOB, age, CDL number, state, years experience, hire date, violations, accidents

🚚 Vehicle/Equipment Schedule

Unit numbers, year, make/model, VIN, vehicle type, GVW, stated value

⛽ IFTA Reports (4 quarters)

Jurisdictional miles, fuel purchases, states traveled, tax paid per state

📉 Loss Runs (3 years)

Claims history, incurred amounts, loss ratios, claim details, carrier info

📋 Motor Vehicle Records

Individual driver MVRs with violations, points, accident history

⚙️ Processing Services

📧 Email Watcher

  • IMAP IDLE push (1-2s latency)
  • Auto-reconnect on failure
  • Marks emails as read
  • Creates submission + starts pipeline

services/email_watcher.py

📄 Document Processor

  • PyMuPDF for PDF text extraction
  • openpyxl for Excel parsing
  • python-docx for Word files
  • Fallback: filename-based classification

services/document_processor.py

🏷️ AI Classifier

  • First 2000 chars → Gemini API
  • Returns type + confidence %
  • Types: application, driver_roster, vehicle_schedule, ifta_report, loss_run

services/document_classifier.py

📊 Data Extractor

  • Type-specific extraction prompts
  • Returns structured JSON per doc type
  • JSON schema validation on AI output

services/data_extractor.py

🔬 AI Analyzer

  • Full submission context analysis
  • Risk scoring (1-10 scale)
  • Cross-document conflict detection
  • Recommendations generation
