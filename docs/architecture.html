Architecture Layers

📨 Layer 1 — Ingestion INTAKE

▶
📧
Email Watcher
IMAP polling service monitors inbox for new submissions from agents/brokers. Extracts attachments automatically.
📤
Web Upload
REST API endpoint for manual file upload. Supports drag-and-drop of multiple documents simultaneously.
☁️
S3 Storage
AWS S3 for durable document storage with server-side encryption. Local filesystem fallback for development.

⚙️ Layer 2 — Document Processing AI + PYTHON

▶
👁️
Vision OCR
Gemini 2.0 Flash Vision extracts text from images (CDL photos, scanned docs). Runs in parallel batches of 3.
📄
PDF/Excel Parser
Python-native extraction using openpyxl and regex. Zero AI cost for structured documents.
🏷️
Batch Classifier
Single AI call classifies ALL documents by content (not filename). Identifies: application, roster, equipment, IFTA, loss run, CDL.
🔍
Data Extractor
Hybrid extraction: regex for Excel/PDF/DL text, AI for insurance applications. Structured JSON output per document.

🧠 Layer 3 — AI Intelligence GEMINI

▶
📋
Summary Generator
Professional underwriting summary covering business profile, fleet, drivers, geographic scope.
⚠️
Conflict Detector
Cross-document analysis for name mismatches, count discrepancies, date inconsistencies.
📊
Risk Assessor
Scores risk factors 1-10: fleet age, driver experience, geographic spread, safety profile.
💡
Recommendation Engine
Actionable recommendations: accept, decline, request info, refer to underwriter.

All 4 analysis calls run in parallel using ThreadPoolExecutor for maximum speed.

📏 Layer 4 — Rules Engine 19 RULES

▶
✅
Eligibility (3 rules)
Vehicle types, deductibles, physical damage coverage checks per Knight Guidelines.
🚗
Driver (7 rules)
CDL verification, experience, age, points thresholds, DUI/criminal history screening.
⛔
Exposure (1 rule)
Prohibited exposure: hazmat, Mexico border, towing/recovery, intermodal.
📑
Submission (3 rules)
Completeness: FEIN/SSN, MC/DOT, loss runs, 4 IFTA quarters.
⛽
IFTA (4 rules)
Fleet MPG reasonableness check per quarter (4.5-8.5 range).
🎯
Selective (1 rule)
Premium per power unit threshold ($13K minimum or 20+ units).

Knight Specialty Insurance — Underwriting Rules

ELIG-001
Target Risk: Semi-Trucks Only
Only semi-trucks hauling general freight. No straight trucks, tow trucks, dump trucks, cranes, or mobile equipment.
CRITICAL
ELIG-002
Ineligible Vehicle Types
Rejects: dump trucks, concrete mixers/pumpers, cranes, tow trucks, mobile equipment regardless of GVW.
CRITICAL
ELIG-006
Auto Liability Deductibles
Auto liability deductibles are not allowed under this program.
INFO
DRV-001
Valid CDL Required
All drivers must hold a valid U.S. Commercial Driver License (CDL).
HIGH
DRV-002
Minimum 2 Years CDL Experience
Drivers must have minimum 2 years CDL driving experience in the U.S. or Canada.
HIGH
DRV-003
Minimum Age 23
All drivers must be at least 23 years old.
HIGH
DRV-004
Age 65+ DOT Medical Exam
Drivers age 65 and older must provide an acceptable DOT medical examination report.
MEDIUM
DRV-005
Max 6 Points in 3 Years
Drivers with more than 6 points on their record in the past 3 years are flagged for review.
HIGH
DRV-006
Max 4 Points in 12 Months
Drivers with more than 4 points in the past 12 months trigger escalation.
HIGH
DRV-100
Unacceptable Driver History
Quick decline for: DUI/DWI (5yr), vehicular homicide/assault, reckless driving, 30+ over speed, hit-and-run, fleeing, felony MV, school bus.
CRITICAL
EXP-001
Hazardous Materials Prohibited
Hazmat cargo, lithium batteries, Mexico border (50mi), towing/recovery, intermodal, waste disposal.
CRITICAL
SUB-001
FEIN/SSN Present
Application must include Federal Employer ID Number (FEIN) or Social Security Number (SSN).
HIGH
SUB-002
MC/DOT Number Present
Application must include Motor Carrier (MC) number or DOT number.
HIGH
SUB-003
Loss Runs Current
Loss runs must be valued within 60 days, including expiring year and at least 3 prior years.
HIGH
SUB-005
4 IFTA Quarters Required
Most recent 4 quarters of IFTA reports must be included in the submission.
HIGH
IFTA-001
Fleet MPG Reasonableness
Fleet MPG should fall between 4.5 and 8.5 per quarter. Values outside this range are flagged.
MEDIUM
SEL-003
Premium Per Unit Threshold
Accounts under $13K per power unit must have at least 20 power units.
MEDIUM

End-to-End Data Flow

📨 Intake

Email arrives → IMAP watcher detects new message → Attachments extracted → Files uploaded to S3 → Submission record created in SQLite with status "pending"

IMAP • S3 • SQLAlchemy
↓
📝 Extract

PDFs → PyPDF2 text extraction • Excel → openpyxl structured parse • Images → Gemini Vision OCR (parallel, 3 workers) → All raw text stored in document records

PyPDF2 • openpyxl • Gemini Vision
↓
🏷️ Classify

First 500 chars of each document → Single batch AI call → Content-based classification (filename intentionally excluded) → Types: application, driver_list, equipment_list, ifta_report, loss_run, drivers_license

Gemini Flash • Batch Processing
↓
🔍 Extract Data

Each classified document → Type-specific extraction → Excel/PDF/DL: Python regex (free) • Application: AI extraction (1 call) → Structured JSON per document → Driver data deduplicated by license number

Python Regex • Gemini Flash
↓
🧠 Analyze

Unified data → 4 parallel AI calls: Summary, Conflicts, Risk Assessment, Recommendations → Completeness check (Python, no AI) → Confidence score calculated

ThreadPoolExecutor • Gemini Flash × 4
↓
📏 Rules

Extracted data → 19 Python rules across 6 categories → Each rule: PASS/FAIL/WARNING/INFO → Results stored with data_used for auditability → Decision: accept (all pass) / refer (warnings) / decline (critical fails)

Pure Python • Instant Execution
↓
🖥️ Present
